## Investigation Steps

1. Upload evidence.
2. Classify evidence type.
3. Parse raw logs.
4. Build structured events.
5. Extract Indicators of Compromise.
6. Apply Sigma detection rules.
7. Map ATT&CK techniques.
8. Calculate risk score.
9. Correlate related events.
10. Create investigation case.
11. Save investigation.
12. Display dashboard.
13. Export report.
